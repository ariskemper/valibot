# minLength

Creates a pipeline validation action that validates a minimum length.

```ts
const validation = minLength<TInput>(message);
```

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `minLength` you can validate, if string has length higher then min length or array has number of items higher then min length. If the input is not a valid min length, you can use `message` to customize the error message.

## Returns

- `validation` <Property {...properties.validation} />

## Examples

The following examples show how `minLength` can be used.

### minLength schema for string

Schema to validate a minLength of a string.

```ts
const minLengthStringSchema = string([
  minLength(3, 'The length of string is lower than the min length'),
]);
```

### minLength schema for array

Schema to validate a minLength of a string.

```ts
const minLengthArraySchema = array([
  minLength(3, 'The length of array is lower than the min length'),
]);
```
